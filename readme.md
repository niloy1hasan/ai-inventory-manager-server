# AI Model Inventory Manager - Server

This is the backend server for the **AI Model Inventory Manager**, a web application that allows users to manage AI models. Users can add, view, edit, delete, and purchase AI models. The server is built with **Node.js**, **Express**, and **MongoDB**.

---

## Features

- Add new AI models with details like name, framework, use case, dataset, description, and image.  
- View all AI models or view specific model details.  
- Edit and delete AI models (only by the creator).  
- Search models by name.  
- Track model views and purchases.  
- Manage user registration and data.  
- Purchase AI models and track purchased models.

---

## Tech Stack

- **Node.js** - JavaScript runtime  
- **Express.js** - Web framework  
- **MongoDB** - Database for storing models, users, and purchases  
- **Cors** - Enable cross-origin requests  
- **dotenv** - Environment variables for configuration
